# ♠️ BJ Counter  
  
一個基於瀏覽器的黑傑克計牌訓練器，實現 Hi-Lo 計牌系統和基本策略建議。 
🔗 **[立即使用 (GitHub Pages 連結)](https://imfuckingwise.github.io/bj_counter)**  
  
## ✨ 功能特點  
  
### 🎯 專業計牌系統  
- **Hi-Lo 計牌法**：業界標準的計牌系統 [1](#4-0)   
- **真數計算**：準確計算標準化真數（TC） [2](#4-1)   
- **優勢評估**：實時顯示玩家數學優勢百分比  
  
### 📊 完整策略引擎  
- **基本策略表**：內建對子、軟牌、硬牌三套完整策略 [3](#4-2)   
- **即時建議**：根據當前手牌和莊家牌提供最佳操作建議  
- **保險決策**：基於真數的科學保險建議（TC ≥ +3） [4](#4-3)   
  
### 💰 科學下注系統  
- **階梯下注**：基於真數的動態下注建議 [5](#4-4)   
- **風險管理**：在劣勢時最小化下注，優勢時最大化收益  
  
### 🎮 實戰訓練功能  
- **多席位模擬**：支持7個玩家席位 [6](#4-5)   
- **快捷輸入**：鍵盤快捷鍵支持，適合實戰節奏 [7](#4-6)   
- **撤銷系統**：三階段撤銷，讓訓練更靈活 [8](#4-7)   
  
### 📈 數據分析  
- **會話記錄**：自動保存遊戲歷史到本地存儲  
- **CSV 導出**：支持數據導出進行深度分析 [9](#4-8)   
  
## 🚀 快速開始  
  
### 安裝  
無需安裝！直接在瀏覽器中打開 `index.html` 即可使用。  
  
### 基本操作  
1. **選擇席位**：按數字鍵 1-7 選擇玩家席位，按 D 選擇莊家  
2. **輸入牌張**：按 2-9、T（10）、A 輸入對應牌張  
3. **標記我的席位**：按 M 鍵標記當前席位為"我的席位"  
4. **新局/新牌靴**：按 R 開始新局，按 N 開始新牌靴  
5. **撤銷操作**：按 Backspace 撤銷上一步操作  
  
### 鍵盤快捷鍵  
| 按鍵 | 功能 |  
|------|------|  
| 2-9, T, A | 輸入對應牌張 |  
| 1-7 | 選擇玩家席位 |  
| D | 選擇莊家 |  
| M | 切換"我的席位" |  
| X | 確認 Double |  
| V | 確認 Split |  
| Z | 切換 Split 手牌 |  
| R | 下一局 |  
| N | 新牌靴 |  
| Backspace | 撤銷 |

## 🎯 算法說明  
  
### Hi-Lo 計牌系統  
- **小牌（2-6）**：+1  
- **中牌（7-9）**：0  
- **大牌（10, A）**：-1  
  
### 真數計算公式  
真數 = 流水數 ÷ 剩餘副數

  
### 下注策略  
- TC < 1：1 單位  
- TC 1-2：2 單位  
- TC 2-3：4 單位  
- TC 3-4：6 單位  
- TC ≥ 4：8 單位
